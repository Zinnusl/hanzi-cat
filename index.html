Hanzi Cat Instructions

Type the pinyin (letters then tone number 1-5) for each falling Hanzi before it reaches the cat's claws. Press Enter to submit. Backspace to correct.

Controls

  • Letters a-z: build pinyin syllable
  • Digits 1-5: add tone number
  • Enter: submit current input
  • Backspace: delete last character
  • Esc: close this dialog

Scoring & Lives

Maintain combos by entering correct pinyin (timing leniency allowed). You have three lives (hearts). Missing a note or wrong submission costs a life.
